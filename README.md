# PublicData


先说开局，不管你打算走什么路线，这个游戏的开局都是差不多的，捏脸/选词条/roll基础数值这些我在之前的攻略中写过，这里不再做复述，不懂的可以翻我之前写过的攻略。

逆天改命的选择

水力四射

剑修的第一个关隘在筑基境选择逆天改命词条上。这里普遍都比较推荐水力四射。剑修之所以要用水力四射，是因为剑修的左键普攻攻击频率极高，更容易触发水力四射。

同时水力四射自带瞄准功能且输出不俗，对于“对群”能力较弱的剑修来说，它是一个非常优质的清小怪功能，弥补了剑修的短板。

可以说它直接颠覆了你对逆天改命的认知，我在重新练新号选择了水力四射后，真的是有被爽到。
￼

但必须注意的是，它最适合的时机是在练气升筑基的时候，再往后就很不建议了。说的直白一点，如果你都已经具灵或金丹了，千万别选水力四射。

要知道虽然水力四射在开局的时候可以让你无比高效地清怪，但随着境界提升它的作用也越发弱化，甚至还会有妨碍视线的负面作用，所以是否要选择，一定得慎重考虑。

最关键的是，水力四射刚刚被吹起来的时候，官方就把水力四射给削弱了一波，强行加入了5秒冷却时间，这就导致水力四射的强度被大幅弱化。
￼

虽然目前只开放到了元婴境界，但可以想象，随着玩家实力不断提升，水力四射的作用就会越发地边缘化。

总之，个人虽然也推荐大家选择练气升筑基的时候选择水力四射，但并不是必选项，如果考量以后的发展，也许选择其他更有用的词条会更好。

武法入门→武法进阶→武法精通

筑基期突破到结晶期，推荐选择武法入门，这是一个可进阶的词条，最高是武法精通，可以减少40%的武技冷却。

注意，这里的武技指的就是你的左键普攻，而减少冷却的意思就是增加攻速。对于剑修来说，主要的输出方式就是靠左键普攻，所以增加攻速就非常关键，就好比LOL里的ADC出了绿叉。
￼
￼

看到文章最开始我放的那段动图了没？如果你没有武法精通，是达不到那样效果的。所以武法入门→武法进阶→武法精通这三个逆天改命对剑修来说是必选项。

顺带一提，武法入门非常难roll，我也是S/L了一个多小时才总算roll出来，大家要有耐心。

兵刃专精

大家在金丹境时，切记一定要点【兵刃专精】。之前我发视频的时候，有小伙伴好奇问我为什么能直接进场开大，就是因为我点了兵刃专精。

但你要注意，这个词条最好是在金丹期的时候点，早了没用，晚了可惜。
￼

为什么这么说呢？因为在练气和筑基期，你能学到的神通只有一个骤天剑。但是它的输出力度是根据剑影数量相关的，如果你提前点了兵刃专精，就算你能进场就放神通，其实也是放了个寂寞，只能看到一道巨剑孤零零的落下。

嗯，这都是血的教训。
￼

而当你升到金丹期的时候，在新门派就可以学到第二种神通万神剑，这个神通是剑修各种神通中，唯一一个威力跟触发条件不相关的神通，也就是说利用兵刃专精这个词条，即便你直接开场放万神剑，也能达到100%的杀伤力。

从设计上，神通就相当于一个角色的大绝招，而兵刃专精让你开场就满气槽，所以万神剑在金丹期的杀伤力无比巨大，在BOSS脸前释放可以让杀伤力最大化，直接秒半行血都不叫事。

所以剑修在金丹期以前，可以靠水力四射秒群怪，获得一个比较舒适的开局，而到了金丹期，则是利用兵刃专精+万神剑的组合，可以非常有效地秒BOSS。

但是万神剑跟水力四射一样，都是随着境界的提高，作用也随之不断弱化的。以前万神剑能直接秒BOSS，而等你到了元婴期，万神剑基本就只是个伤害略高的技能而已了。

所以升金丹期选择兵刃专精是这个词条最得风得水的时候，当然后期这个词条也非常关键，而不像水力四射那样变得可有可无。
￼

最后我给大家总结一下剑修逆天改命的路线：

筑基期：水力四射（可选）

结晶期：武法入门

金丹期：兵刃专精

具灵期：武法进阶

元婴期：武法精通

千万别选什么小/中/大李飞剑，剑灵入门，选错真的毁存档，这也是血的教训。

武技/绝技/身法/神通的选择

这里我必须说一句，目前游戏只开放到了元婴期，而在元婴期之前的各个境界，我个人是不太推荐大家去精细化升级某一项技能的，连参悟都没必要。因为下一个境界的技能强度必然要强于上一个境界，所以只要进入新的境界，就立刻去学这个境界的技能，对你的提升帮助最大。

往往我都已经进入下一个境界了，技能经验还没升个几级。

当然，身法和心法除外，它们对你的影响不是很大，而且心法在早期还有道点的限制，可以不用跟着境界提升。

不过，若是你玩的是最高难度，觉得当前难度实在不好打的话，那么还是有强化技能必要的。

如果你选择了水力四射，那么在元婴境之前，我推荐大家左键普攻选择0.2秒攻速的孤星剑或者0.3秒攻速的清风剑，这样可以让水力四射的触发频率最大化。

个人更推荐清风剑，虽然攻速慢于孤星剑，但孤星的准头是真的差。随着水力四射的作用越发下降，普攻伤害就显得越发可观，这时候你就会发现还是清风剑更好用，而且清风剑往往自带穿透词条，最适合对群的状况。
￼

到了元婴期之后，无脑选月影剑就对了。

别看它攻速慢，威力低，还会被建筑物阻挡，但可以同时发射至多六枚飞剑就是牛逼，单论输出效率是所有剑修武技中最高的。品质越高越好，这样基础伤害越高，参悟出的高级词条也越强。

词条最好是叠流血buff的，因为这类词条可以减身法CD，有助于为后期的加特林打法续航。
￼

绝技推荐落日剑，没别的原因，就是拿来清小怪的。

等你玩到元婴期，水力四射基本就废了没啥大用，而这个时候你清小怪就得靠落日剑来清了，它是范围伤害，可以一次攻击多个目标，尤其是在下副本遇到三四个小BOSS齐出的时候，落日剑就是要比其他绝技更有针对性。

不过落日剑的问题也比较明显，打不着移动能力强的敌人，比如说别的修仙者。输出也比较水，打BOSS非常刮痧。

所以个人建议优先选择有减少身法冷却时间的落日剑，让它也能发挥一些余热。或者选择减消耗词条多的，缓解灵力消耗的压力。
￼

神通依然选择万神剑，但这里务必要注意，看看下面的词条里有没有减武技冷却时间的。

因为到了元婴期，万神剑本身的伤害就不算太高了，但关键是它有一个词条，是在万神剑持续期间可以大量的提高你左键普攻的攻速！

所以你看到文章开头我用月影剑可以打出那样的效果来，就是因为我进场就直接开了万神剑，导致武技攻速大幅提升，于是剑修就被我玩成了加特林，哪怕是当前副本最强的BOSS也能直接瞬杀。
￼

剑修的身法选择方式跟万神剑一样，主要就看词条里有没有增加普攻攻速的。

剑修的身法用处都点在了增攻速上，所以如果你没有这个词条，那么你与其用剑修的身法，还不如换别的身法对你更有帮助。

所以剑修的正常爆发开局就是进场直接怼到BOSS脸前，然后依次按下鼠标右键→R键→空格，这样就能让剑修的左键普攻攻速达到最大化，于是就有了本文开头的那个动图。

心法的选择同样要看词条。主要就看减身法冷却时间的、减神通冷却时间的，提高身法持续时间的诸如此类等等。

因为你要知道，剑修玩到后期就只有一个目的，尽可能提高月影剑的攻速，以及开加特林的频率和持续时间。

怎么提高攻速？你就开万神剑和身法啊。所以要么提高身法持续时间，要么减少冷却时间，就能变相提高剑修的实战强度。
￼

总结：

剑修是一个前中期乏力后期非常变态的一个职业，但如果你选错了功法甚至是选错了逆天改命的词条，剑修也得被玩废，我的第一个档就是这么被废掉的。

剑修玩到后期，就是用尽一切手段来提高普攻的攻速，拿它当一个ADC来玩，以此让月影剑的输出效率达到最大化，这种优势越到后期感觉越明显。

当然剑修这种爆发打法也非常耗蓝，我一般只有打BOSS才用，平时下副本开个万神剑就行了。不过后期也基本不怎么缺钱，把天/地道之息卖拍卖会，分分钟上百万，拿来买回蓝药绰绰有余。
