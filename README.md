
1. 战斗类逆天
2. 属性类逆天
3. 非战斗类逆天
大致将逆天改命分为这三类，首先讨论战斗类逆天在洪荒难度下的实用性和突破做选择时优先程度的建议。实用性考虑两个方面，一是筑基期开始到元婴期过程里，角色成长期的收益，二是元婴毕业，成型期后是否还有较大增幅。实用性评价分为强烈推荐，较为推荐，中立，不太推荐，无用五档。优先程度依据不同职业自然选择不同，优先度评价分为T0到T3(T0：出现时候的最优先选择，T1：泛用性很强或某些流派的首选，T2：没有什么好选择下的选择，较鸡肋，T3：娱乐天赋)
1. 妖术系列
成长型逆天，入门只能变羊(实际是鸡)，进阶及精通变羊后敌人每过几秒会随机方向移动，并在移动后向附近单位攻击(不分敌我)，造成的伤害基本和未变羊前差不多，攻击变羊敌人伤害略有增加(怀疑是防御暂时变零，也不确定)，精通阶段是一个小范围aoe效果(大概和刀修半月刀攻击范围差不多大)群体变羊。最大问题不能对修真者和大妖兽使用，洪荒难度下目前只能沦为娱乐向逆天。实用性：无用，优先度：T3
2. 菊花系列
成长型逆天，战斗中移速提升15%(入门)到20%(进阶后)，前两段带负面效果(背面承伤增加)精通后全正面效果(背面承伤减伤)且进阶后带有概率喷火伤害(用元婴档进图挨打试了下进阶和精通分别造成1000和1500左右伤害)，聊胜于无。重点在移速加成，前期400以上移速按菊花烫15%加成算也有至少60点速度增益，相当于带来了两条红色移速词条的收益。
后期角色移速普遍达到500以上(剑修带剑法资质加移速词条神功基本可以过600)的情况下，20%等于100点以上的移速加成，超过三条红色移速词条提供的速度。高移速意味着躲boss技能在你眼中从困难模式直接变为时停模式，九尾狐的魅惑刚放到一半角色已经绕道boss背后去了，且在菊花系列的加成下，纯剑修(剑身法神功提供高额移速加成)甚至可以和山挥道友在秋名山上一起飙车，剑身法的减伤和无视障碍物让山挥根本追不上你。对所有职业来说移速收益都是相当高的，度过了前期之后仅次于攻击属性(因为根据贴吧大佬测试每个阶段防御有上限阈值)，而前两阶的负面效果相比之下则影响细微。实用性：强烈推荐，优先度：T1
3. 跟屁虫系列
成长型逆天，进阶前后伤害不变(元婴测试固定伤害200多)，区别在aoe范围逐渐增加(大概是从精英怪技能红圈小圆到中圆的感觉)，攻击频率目测也没变。神奇的一个系列，记得在游戏刚发售不久坊间盛传虫子最后会化龙，因为版本还未更新所以对于这个逆天还不能最终定论。测试时候发现一个有趣效果，虫子击中敌方单位时会给他们施加一层左键buff(需要左键带30%概率的对敌方施加型buff)，打到多少敌人都各上一层，没打中无法施加，以跟屁虫的攻击频率看无法往上叠加最多只能一层左右且不稳定因为跟屁虫攻击位置很随机且有最大攻击距离。目前能想到的是对需要前置条件是对敌方施加buff杀死的神通起到一丢丢帮助(神火核，万神剑)，但是因为虫子基础伤害过低，这样更像彩蛋一样的特殊效果收益还是甚微。实用性：不太推荐(不是无用的原因在于至少还有一个上buff作用和更新后或许会鱼跃龙门)，优先度：T3
4. 小宝的盾系列
成长型逆天，进阶后增加抵挡5次技能后施放振波伤害带减速(中宝)击退加减伤(大宝)，护盾大小距离没有成长变化，伤害的话元婴期爆一次大概1500和1800左右，大宝的盾在施放振波伤害时可以将范围内弹幕技能清空中宝试了好像不行。(测试时九尾狐的全屏狐群被清空了一半)可以抵挡某些敌方有弹幕路线的技能(试过可以抵挡的有狐狸的攻击，鸣蛇的火球，雷神雷电好像挡不了，当康技能也挡不了，山挥没重生估计丢石头应该可以档)，没有弹道的技能则不行(比如蘑菇怪的蘑菇轰炸，蟾蜍喷烟等等)，是显而易见的防御型逆天，但是因为盾牌是围绕角色不停旋转的，就会导致有些时候想档技能挡不住的情况出现(当然高手可以掌握盾牌移速规律主动上前抵消掉boss弹道类技能)，还有一个弊端是据说盾牌也会挡住某些自己的技能。(因为从没选过盾系列，测试时使用剑修木修左键不会被挡住，有经验的道友可以分享下)。
因为能挡住的技能种类有限，而且抵挡效果也比较随机，总体实用性较低。实用性：不太推荐，优先度：T2
5. 小李飞剑系列
成长型逆天，伤害随境界提升增加，基本微不足道，中李开始具备控制。系列最大特色是放出随机的控制技能，在每次使用左右键，身法，神通时飞剑都有概率(左键大概三到五下会飞出，没打到怪对空气a也会)飞出造成伤害和控制，大李更是每10次飞剑后造成群体随机控制，且不耗蓝。
看起来很优质的逆天，但是最大缺点在于鬼谷boss控制机制问题，游戏的boss每次吃到一个控制技能后会跳过一个技能阶段(例如开明兽要发buff的前摇出现时，用带控制右键技能可以打断掉)，而中李开始飞剑随机发出中毒、晕眩、减速、燃烧、冰冻、**、吸血、击退、灵力损失、缠绕、沉默以上11种效果，其中晕眩冰冻**缠绕沉默算硬控可以造成打断使boss跳过一个技能阶段，而其他不行(击退目测不行)，这就会导致在某些boss战中，带中李或大李的玩家会遇到接连跳过数个技能然后直接二连发离谱连招的boss(例如碎梦山挥跳过全程，直接二连全屏带旋风冲撞，满屏幕都会是风，还有九尾狐连跳几个技能，二连全屏狐群)，以及有时候boss被控制中途由于飞剑下一个并非硬控使boss突然恢复并光速使用下一个技能(例如连续打断尸傀技能，跳过几个阶段后boss醒来直接封印技能突脸)。
对于新人洪荒玩家，尤其是还不太熟悉boss连招有可能卡在某些boss战前的，不太推荐选择小李系列，对于洪荒老手来说自然是随意选择。当然大李飞剑在元婴成型后仍然是有收益的逆天，对已经成型的0cd流派来说，大李附带的控制变成了接近无限持续。总结来说瑕不掩瑜还是比较有价值的逆天系列。
实用性：元婴前中立，毕业后较为推荐，优先度：T1
6. 四射系列(水力，火力，雷力)
三个单独的逆天，放一起讲吧。三个都是输出型逆天，面板伤害随境界提升增加，实际战斗中伤害与角色攻击力挂钩(攻击力越高，四射实际伤害越高)，其中水力火力每次使用左右键，身法，神通技能都有概率施放(不管有没有打到怪，空放也有概率施放)，水力攻击距离至少可以达到全屏3/4(测试环境有限，没把boss拉到地图对角极限，有经验的道友可以分享下)，火力攻击距离就和火修左键豪火术差不多，雷力是围绕角色固定范围的闪电球。对应灵根不增加四射的伤害，但是灵根属性过低四射会概率被怪化解(以元婴期5星救人任务怪物做测试时，150及以下水火灵根概率被小怪化解，200以上小怪精英怪基本无化解，boss350灵根下依然概率化解)。
输出方面火力单发伤害高，但存在攻击距离较近和频率低的不足。水力单次触发大概持续5秒左右，可以叠加使得时间持续，单次滋水枪攻击次数35次左右，在未满足0cd的情况下(武法精通40%加刀大法25%)已经可以保持水力全程持续，综合伤害，触发难易度都远超其余两种四射，尤其对于功法6种职业来说，虽然四个技能都有概率触发水力(火力)四射，但功法职业吃减cd红利更多也更容易，左键越快意味着四射输出越高(65%减cd下水力全程存在)，洪荒难度下无论成长期还是成型期都是很推荐的逆天(尤其前期能拿到水力收益更高，理由是水力具有基础伤害，在前期角色伤害低的情况下大幅增加dps)。火力因为前面所述的距离和频率问题存在明显短板，不太适合远距离职业拿，近战系和一些选择中距离左键的玩家没有更好逆天选择的情况下可以拿。雷力因为要求绝技触发以及攻击形态的问题，基本无用了。
水力 实用性：强烈推荐，优先度：T0或T1(取决于是否选择左键减cd的搭配功法)
火力 实用性：中立，优先度：T2
雷力 实用性：无用，优先度：T3
7. 鬼修系列
成长型逆天，首先逆天的描述文字有误，入门召唤的是尸傀弓箭手不是尸傀战士，其次召唤物继承了角色防御，血量不知是百分多少，入门最多2只(进阶3只，精通4只)，击杀单位后概率召唤，能充当肉盾的作用其他就没了。攻击力非常低，低到我以为哪里出问题了，面板1300多攻击的角色召唤的弓手和战士，能打20多伤害差不多。尸傀法师远超同僚，能打出100多伤害(暴击200多)，且防御力比角色本体高，可惜血量都太少。这样离谱的低攻击力我觉得要么是数据有误，要么就是纯娱乐天赋了(居然还是成长型占三次逆天的改命)。
实用度：无用，优先度：T3
8. 黄巾天师
洪荒难度外合格的优质肉盾，继承角色防御，血量疑似100%本体血量，体积大能挡住相当多的boss技能，但也因为体积过大经常会吃到boss所有技能暴毙。洪荒难度下黄巾难活过boss一轮技能循环，由于召唤条件限制，一场战斗只能召一个黄巾力士，因此沦为鸡肋选项。攻击力方面和鬼修召唤物接近，能打出两位数输出，攻击范围是近战小型aoe伤害。索敌机制是向最近的敌方单位移动攻击，经常会打假赛被小怪缠住，远离了boss连挡枪职能都无法履行。
实用性：不太推荐，优先度：T2
9. 武神残诀
强力逆天，一拳超人bd的必备组件，对刀修来说也是优先度极高的选择。对其他职业来说泛用性不是很强，但拿了也都有一定收益。效果简单粗暴就不多赘述了，转化的护盾量至少有30%，在长时间刷怪的时候可以省一些丹药。
实用性：较为推荐，优先度：T0(刀拳)或T1
10. 青帝遗录
竟有如此红色逆天，角色死完后召唤5只花妖，攻击力两位数，防御力大概和角色差不多吧，没什么意义因为角色死掉后靠它们刮痧不可能刮死boss。在之前洪荒难度没有双鱼佩点数的版本，可以在手速慢没及时按alt+F4的时候提供多一点反应时间，现在的版本完全沦为娱乐天赋(甚至不如其他娱乐天赋)。
实用性：无用，优先度：T3
11. 血魔系列
成长型逆天，吸血效果及距离随进阶增加(血魔300距离和刀修左键的半月刀旋风刀基础距离一样，血魇也只到400距离)。从这两个属性来看好像是刀修专属逆天，但因为当前版本只有5个逆天改命，刀修有优先度更高的武法系列和武神残诀需要选择，且吸血效果有刀大法提供的高额加成，因此对刀修来说，血魔系列更像是锦上添花的非核心组件。对拳修来说这个逆天的距离和吸血效果应该更为适合(等待拳修大佬分享经验)，对其他职业左键选择了中距离射程的玩家，因为是全能吸血(左右键身法神通造成的伤害，只要在距离内都可以吸血)反而可能会起到一些意想不到的用处。
实用性：中立(占三个逆天改命有所降分)，优先度：T1.5(刀拳)或T2
12. 疯魔战意
不是很确定现在有没有实装的逆天，印象里好像没有见过，搜了下吧里讨论的也很少，突破有roll到的可以分享下。很有意思的一个逆天，承伤减少50%，增加伤害100%(增伤效果打boss时候明显)，负面效果是wasd移动键不受控制人物会随机方向乱走。触发条件需要低于20%血量，因此和武神残诀是最优搭配。因为人物不受控制随机乱走容易吃技能，所以装配风身法的风剑，风修可能比较合适，可以形成半自动打怪。如果使用疯魔加武神的话，尽量不要带回血词条的经和其他一些带回血词条的心法。
实用性：不太推荐(需要搭配武神的前置要求过于苛刻)，优先度：T2.5
13. 专精系列(五个免除神通前置条件的逆天)
万金油型逆天，除了指修其他系都可以搭配。虽然存在一些很容易能够达成前置条件的神通，但是选了总是不亏的，技能早用早cd，每场boss战说不定就能多放几次了呢。对于一些bd来说是较为重要的逆天(水剑高速启动用的万神剑，枪系为主体的鬼域枪)。
实用性：较为推荐，优先度：T1
14. 红尘剑魂系列
成长型逆天，稀少的只有两阶的成长逆天，造成200/400次伤害后施放神通触发，1.密卷带进房间随机敌人施加掉血buff的词条造成的buff掉血每秒掉血算一次;
2.带多段攻击的左键(如月影剑)每段打到敌人都单独算一次
3.水分身造成的二次进场叠加buff不算在内，同时分身打的伤害也不算次数;
4.召唤物的攻击也不算次数，但是召唤物死完后形成的伤害(比如木修功法使小花死后形成沼泽，沼泽造成的伤害)算次数;
红尘剑匣这个逆天我记得在游戏发售初期很受追捧，因为名字和特效都很帅，测试下来伤害依然和角色攻击面板挂钩，面对元婴期5星任务boss(112万血的九尾狐)剑匣的伤害拉跨，人物1300攻击面板的前提下造成4万左右伤害，剑魂的表现好很多，造成大概17万伤害，纯以数据来说一次性打掉了boss15%的血量还是比较可观的(至少不是刮痧)，且一共只占2个逆天，在没有遇到更好选择的情况下值得一选，毕竟是实打实的造成伤害。
实用性：较为推荐，优先度：T2
15. 血祭系列
成长型逆天，入门只有血爪一段伤害，进阶及精通血爪造成伤害后会产生血池，血池上的敌人持续debuff伤害。
技能伤害与人物攻击属性高低和最高功法资质以及平均功法资质(说明有误)都相关，血爪攻击的位置和距离与左键攻击位置和距离一致(因此会空技能，同时选择远程左键的时候更有一点优势)，在简单环境测试下面板攻击1200，全功法资质350的人物打5星宗门任务boss(93万血九尾妖狐)打满伤害的情况下(血手加血池)大概是2万多，同时分别试过降低500攻击和5种功法资质为1，1种功法资质350以及全功法资质为1的情况，结果是第一第三种情况伤害大幅降低(大概每次输出都差了1万多)，第二种情况略降几千单轮伤害。控制变量的大概结论是平均功法资质的影响相对较小，面板攻击低以及最高功法资质低的情况对伤害影响很大。精通带血池回血但是很少几乎可以忽略不计，逆天的面板伤害及施放所需血量消耗随境界提升增加。伤害方面要考虑打的是站桩型boss还是带位移突进技能的boss，面对不同的类型，血祭能打出的伤害差别较大(血池debuff伤害基本占总伤害一半以上)。
综合来说远程职业更适合这个逆天，和更适合中近距离职业的血魔来说正好相对，此外刀修和带吸血左键的流派也可以选择(抵消扣血负面效果，纯增加输出收益)。
实用性：中立(视面对不同情况效果差别较大)，优先度：T1.5(远程左键职业以及左键吸血职业)或T2
16. 归一诀
成长型逆天(2段)，受到50/100次伤害后施放神通主动激发使用，获得一个持续15/30秒的减伤80%/90%效果。实测效果较好，减伤效果是实打实的减少了boss技能的80%和90%伤害，意味着在持续时间内可以硬抗boss伤害打满输出，且可以自主控制什么时候使用也是加分的点，使用的效果加上可控机制，在防御类逆天中建议优先考虑。
实用度：比较推荐，优先度：T2(确实算是比较实用的逆天，但可惜的是因为游戏机制原因大部分攻击型天赋都会排在较靠前的位置考虑)
17. 星耀尘寰
成长型逆天(2段)，击杀敌人25/50个后施放神通主动激发使用，持续时间(15/30秒)内有一个围绕角色身边的雷属性aoe伤害(范围较小)，且靠近会致盲敌人(boss被致盲后技能随机方向乱放)和四射系一样灵根不增加伤害但是灵根过低会大概率被化解属性伤害。伤害方面不太高，面对元婴期城镇5星救人任务boss(112万血的九尾狐)打满30秒大概造成2万多伤害。致盲是一个比较有趣的特效，只是不是硬控，boss还是有一定概率会朝着角色所在位置放技能。而且升级后施放的前置条件有些苛刻，带小房间的boss关卡还比较容易达成，野外boss的话就等于要杀好几轮招的小怪才满足条件。总的来说更偏娱乐性逆天。
实用性：不太推荐，优先度：T3
18. 急速代谢
拉丹，很有创意的一个逆天，效果大家应该都清楚。普通使用的情况，在前中期需要过多个小房间才能打boss的时候大幅提升了续航，同时也是一个比较看脸的技能(有时候5个丹一个没回来)。更高阶的理论使用方法就是构筑了流水剑的大佬创造的急速爆破(详细的内容可以去原贴中浏览)，大致上就是依靠水剑身法减右键cd，右键使用引爆流血buff减身法cd形成循环，搭配上带每次使用右键减丹药cd，减少丹药冷却cd这样的录，可以使角色在短时间内连续使用攻击丹药大幅增加攻击力打输出(根据大佬的测试攻击药buff可以叠加)。
此外并不局限流水剑，像刀右键的血刃系带只有一个敌方单位时减cd和减cd词条的，也能达到短时间内不断刷新右键的效果(再加上刀法减少右键cd)，和拉丹逆天具有一定的适配性。大家也可以多去寻找其他系的右键或心法中是否有类似效果的。总结的话急速代谢前中期具有泛用性，对某些流派成型后的伤害也能提供理论上的大幅提升，兼具娱乐性和实用性算是一个较优质的逆天。
实用性：比较推荐，优先度：T1(对想构筑流水剑或cd刀或其他能适配的流派玩家来说)或T2
19. 双灵系列
成长型逆天(3段)，效果简单明了，耗蓝是比较头疼的问题，对灵根职业来说是优先度较高的选择，尤其是召唤系(土木以及神通走神火核的火修)有没体验差比较多，此外还适合其他一些对逆天需求不高的职业。因为很多类型的技能都无法双灵效果，加上升级后总体效果反而不如双灵阶段，而如果一直保持双灵的话则会导致每次突破逆天都有概率出现三灵(卡逆天池)，综合来说只适合一些特定职业，泛用性不是很强。
实用性：中立，优先度：T1(召唤系)或T2.5
20. 蘑灵仙力
成长型逆天(2段)，娱乐天赋中的佼佼者，人物死完后变身蘑菇怪，脚下乘云(持续时间内不会受到攻击)攻击模式也变成蘑菇轰炸，持续时间结束后回复一定血量。升级后战斗中每过一段时间会在地上生成蘑菇(就是蘑菇怪种的，随机变大或变小蘑菇)，角色变大后攻击力显著上升但是移速变慢，变小后移速急速提升(就像吃了蘑菇的山挥道友一样)但是视野缩小，两种变化后防御方面没有变化。注意要先吃过打倒药灵后掉落的奇怪的蘑菇，才有概率后续突破时出现这个逆天。彩蛋一样的逆天，但是升级后的种蘑菇效果也是有一定实用性的(尤其是变小菇)，暂时不知道种出来的蘑菇能不能被召唤物吃掉(等有经验的道友分享)。等于变向给角色增加了血量，拉高容错，娱乐逆天中的优选。
实用性：不太推荐，优先度：T2.5
21. 盾灵系列
成长型逆天(3段)，进阶及精通带10秒的200%反伤效果，和蘑灵，潜能，金蝉系列同样为濒死保命效果的逆天。在越阶和修仙者战斗中效果极佳，对拳修来说是排在武神残诀之后的较优选择。因为保命效果的存在，具有一定的泛用性，在四种同类型逆天中排在较前考虑的位置。
实用性：中立，优先度：T2
22. 潜能系列
成长型逆天(3段)，四种濒死保命效果的逆天之一，附带的效果加移速，加伤害和减cd都很好，但最大问题是只持续5秒实在太短，因此在四种同类型逆天中排在较后考虑的位置。
实用性：不推荐，优先度：T3
23. 金蝉系列
成长型逆天(3段)，四种濒死保命效果的逆天之一，人物濒死后变成土系身法形态移动，并在原地创造一个水分身(比通常的水分身强一些)，综合效果要优于潜能系列一些。
四种逆天横向对比来看，升到顶级后的回复血量：潜能(80%)>金蝉(75%)>盾灵(70%血量的护盾)>蘑灵(50%)
其他特效的实用性来看：盾灵的反伤 > 金蝉的水分身，蘑灵的种蘑菇 > 潜能的buff效果(效果很好但持续时间实在太短)
综合评价四种保命型逆天的选择优先度是盾灵>蘑灵(两段进阶省一个逆天格子)，金蝉>潜能
